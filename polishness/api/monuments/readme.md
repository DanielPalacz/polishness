# polishness - Monuments - tools

## DB SETUP

```
 - From Project DIR:
   PYTHONPATH=. python polishness/api/monuments/tools/db_setup.py --help
 
 1. Through project static directory:
    (should be created first)
    PYTHONPATH=. python polishness/api/monuments/tools/db_setup.py
    
 2. With custom settings:
    PYTHONPATH=. python polishness/api/monuments/tools/db_setup.py -s <schema> -l <db_loc> -f <file_loc>
```
